## ❓ Troubleshooting

If operon does not open:

- Make sure the download finished
- Try opening it again as the current user
- Check that Windows did not block the file
- Restart your computer and try once more

If the app opens but files do not load:

- Check the file path
- Move the file to a simple folder like Documents
- Make sure the file is not locked by another app
- Try opening a smaller file first

If AI features do not work:

- Check your internet connection
- Review your AI settings
- Make sure the service key or account is set up
- Try again after restarting the app
